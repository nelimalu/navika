# this is navika!

A simple CLI-based heatmap habit tracker.
Inspired by James Scholz and [LifeOfDiscipline](https://www.lifeofdiscipline.com/).

Here is a list of all commands:
  1. `view <habit>`
  2. `log <habit> [amount] [date]`
  3. `create <habit>`
  4. `delete <habit>`
  5. `help`
  6. `list`
  7. `info <date>`
  8. `quit`
